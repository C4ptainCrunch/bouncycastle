Bouncy Castle

Bouncy castle is a self-hosted converter to convert Youtube videos (or Soundcloud musics) to mp3. It's like youtube-mp3.org but on your machine (and also, it does not block some videos like youtube-mp3).

It is a simple web front-end to the CLI youyube-dl python converter.

Roll your own !

Bouncy Castle is super easy, try it at home ! It has only few dependencies : python2.7+ and youyube-dl (witch uses ffmpeg)

Install

git clone git://github.com/C4ptainCrunch/bouncycastle.git
virtualenv --distribute --no-site-packages ve 
source ve/bin/activate
pip install -r requirements.txt

Check config.py for an example configuration file and put the values you want to overwrite in production.py

Run

You should run web.py for the web interface and convert.py to convert the videos. (I use supervisord to run both)
